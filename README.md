# Longitudinal Analysis of Cognitive Aging (HRS)

This repository contains a comprehensive analytical pipeline developed in **R** for investigating cognitive trajectories using the Health and Retirement Study (HRS) dataset.

## Research Highlights:
- **Data Integrity:** Cohort construction and rigorous cleaning of large-scale longitudinal survey data.
- **Advanced Modeling:** Implementation of **Linear Mixed-Effects Models (LMM)** with random intercepts and slopes to capture individual aging patterns.
- **Bias Correction:** Utilization of **Inverse Probability of Censoring Weighting (IPCW)** to address potential attrition bias in long-term studies.
- **Visualization:** Predictive margins and standardized trajectory plots.

*Note: Scripts are designed for use with HRS restricted/public datasets. Raw data is not included due to licensing.*
